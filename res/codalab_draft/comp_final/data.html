This is the data for the competition. It is to be used responsibly.

Submission

The participants are asked to submit their data as a csv file named dev.cvs for the development phase and test.csv for the challenge. The csv file has to be zipped for the submission to the system and follow the following format:

movie_name,file_name,goodforairplanes
Belle de Jour,Belle_de_Jour,1
Big Game,Big_Game,0
Birdman Or (The Unexpected Virtue of Ignorance),Birdman__Or_(The_Unexpected_Virtue_of_Ignorance).mp4,1
